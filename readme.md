# storeapp-backend
Set of web services consumed by storeapp front-end application

# Run
The file .env contains all the parameters that the app needs to execute.
```
DB_USERNAME=shop
DB_HOST=localhost
DB_DATABASE=shop
DB_PASSWORD=shop
DB_PORT=5432
DB_DIALECT=postgres
PORT=4000
TEST_HOST=http://localhost
```

## Run app
npm run

# Test
There is a set of test built on jest.

## Run test
npm test
